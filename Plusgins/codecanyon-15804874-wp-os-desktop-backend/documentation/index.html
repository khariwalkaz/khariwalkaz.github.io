Installation How to Install the plugin

1. Install Plugin

Here's a brief information about how to install this plugin

1.1) Upload the plugin

Upload the folder ./WP_osBackendDesktop/ on your server with ftp, in the plugins folder : ./wp-content/plugins/

1.2) Activate plugin

Login into your wp administration, then go on the "Plugins" page. You should see "WP O.S Backend Desktop" in the list. Click on "Activate", under the title of the plugin.

Use How to use the plugin

2. Use the plugin

Here's a brief information about how to use this plugin

2.1) Windows

Once the plugin activated, when you click on a link in the wp backend menu, the corresponding page is now loaded in a window.

You can move your windows in drag & drop, and resize it clicking on a corner and moving the cursor.

In the right corner of the window header, you can see 3 icons.
The first one allows you to enable/disable the fullscreen mode for this window.
Clicking the second one, the window will be minimize. You can open it clicking on the corresponding window button at bottom of the screen.
Click the last icon to close the current window .

To display two windows side by side, simply drag the first one to the left border of the screen, and the second one to the right border of the screen .

2.2) Desktop shortcuts

Each WP user can create its own shortcuts on his desktop. To do that, simply drag and drop a link from the WP menu to the desktop background.
To open the window corresponding to a shortcut, simply double click on it .

When you dragging a shortcut, you can see a red panel containing an icon of trash at bottom of the screen. To delete a shortcut, simply drag it on this red panel.

2.3) Keyboard shortcuts

You can quickly navigate through the windows using this keyboard shortcuts :

  • ALT+S : show the next window
  • ALT+SHIFT+S : show the previous window

You can edit these two shortcuts directly from the settings panel .
